### 2. 변수와 자료형
###### (1) 변수명 규칙
- 문자, 숫자, _, $만 사용 가능
- 숫자로 시작하면 안 됨
- 대소문자 구분함 (age ≠ Age)
- 예약어(예: int, class)는 사용 불가

###### (2) 자료형
- int num = 10; //정소
- double d = 3.14; //실수
- char C = 'A'; //문자
- boolean flag = true; //논리

###### (3) 형변환
- int i = (int) 3.7; //명시적 형변환
- double d2 = 5; //자동 형변환
- Integer.parseInt(); //문자열 -> 정수 변환

###### (4) 문자열 비교
equals()로 비교
== 는 주소 비교

### 3. 연산자
###### (1) 우선순위
산술 -> 비교 -> 논리

###### (2) 연산자

### 5. 반복문
###### (1) for문
```
for(int i=0; i<5; i++) {
  System.out.println(i);
}
```

###### (2) while문
```
int i = 0;
while(i<5) {
  System.out.println(i);
  i++;
}
```

###### (3) do-while문
```
int i = 0;
do {
  System.out.println(i);
  i++;
} while(i<5);
```

### 7. 매서드
###### (1) 매서드
매서드를 사용하는 이유 : 반복되는 코드를 재사용하고, 프로그램을 구조화하여 가독성과 유지보수성을 높이기 위해 사용
```
public static int sum(int a,int b) {
  return a+b;
}
```

###### (2) 호출 및 결과
```
int result = sum(3,4);
-----------------------
7
```

###### (3) 매개변수와 반환
- 매개변수 : 입력값
- 반환 값 : 호출한 곳에 결과 전달

### 8. 클래스와 객체
###### (1) 클래스
객체를 만들기 위한 설계도
```
class Person {
  String name; //속성(필드)
  int age;
  void introduce() {
    System.out.println(name + ", "  + age);
  }
}
```

###### (2) 객체
설계도로 만들어진 실제 인스턴스
```
Person p = new Person(); //객체 생성
P.name = "홍길동";
P.age = 30;
P.introduce(); //매서드 호출
```

###### (3) 생성자
객체 생성 시 초기값 설정
```
Person(String name,int age) { //생성자
  this.name = name;
  this.age = age;
}
Person p2 = new Person("윤",25); //초기값 지정
p2.introduce();
}
```

### 10. 사용자에게 입력 받기(Scanner)
```
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in); // 입력 받을 준비

        System.out.print("이름을 입력하세요: ");
        String name = sc.nextLine(); // 문자열 입력

        System.out.print("나이를 입력하세요: ");
        int age = sc.nextInt(); // 정수 입력

        System.out.println("이름: " + name);
        System.out.println("나이: " + age);

        sc.close(); // Scanner 닫기 (권장)
    }
}
```
